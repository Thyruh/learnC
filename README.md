# learnC

Small C exercises and projects.

The repository is split into chapters with no particular order that gradually introduce:
- syntax and control flow
- functions and recursion
- pointers and memory
- structs and file I/O
- algorithms and larger projects

Each directory contains tasks to implement.

Workflow:
1. read the tasks
2. write the code
3. compile it manually
4. debug it
5. repeat

## Structure

```txt
0_ explore       basic experiments
1_ minicalc      parsing and expressions
2_ controlflow   loops and conditions
3_ funcs         functions and recursion
5_ memory        pointers, malloc, realloc
6_ structs       structs and data layout
7_ fileio        files and CSV
8_ algorithms    search, sort, recursion
9_ projects      larger standalone programs
10_ tools        utility programs
```

The final exercise is to split previous programs into modules and write a Makefile.
